# Polynomial Console Graph

### Graph Polynomials in your Terminal

### To Run

- Build by running `make` in `src/`
- Run the executable `PolynomialConsoleGraph`
- Follow the prompts
- Have fun with ASCII graphs
- Rinse + Repeat
